-Sirve para añadir CSS a JS de manera que no tengamos el css de forma global. Si se tiene un gran archivo de 18.000 lineas del que depende un proyecto, eliminar algo de ese archivo (porque ya no se use) resulta dificil que alguien se anime a hacerlo. Para solucionar esto esta "styled components".
